A unique, horizontal-scrolling gallery that treats every project like a feature film. Each card includes:

*   **Dark Mode Optimization:** Deep black backgrounds with neon emerald accents.

A specific, integrated contact form powered by EmailJS.
*   **Field Validation:** Ensures data integrity.
*   **Success/Error States:** Visual feedback for the user.

## ğŸš€ Getting Started

Follow these steps to run the portfolio locally on your machine.

### Prerequisites
*   Node.js (v18 or higher)
*   npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/asif-mahamud-shaon/portfolio.git
    cd portfolio
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Run Development Server**
    ```bash
    npm run dev
    ```

4.  **View locally**
    Open [http://localhost:3000](http://localhost:3000) in your browser.
