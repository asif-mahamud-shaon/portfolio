# Portfolio - Asif Mahamud Shaon

A premium, cinematic personal portfolio website built with Next.js, Tailwind CSS, and Framer Motion.

## Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion, GSAP
- **3D/Graphics:** Three.js (Optional integration ready), Lucide React Icons

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open Browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## Features
- **Hero:** Interactive grid background, typing effect, floating cards.
- **Identity:** Tabbed card interface for personal branding.
- **Skills:** Orbiting constellation visualization of tech stack.
- **Projects:** Horizontal scrolling cinematic showcase.
- **Experience:** Vertical glowing timeline.
- **Mindset:** animated process flow.
- **Contact:** Minimalist functional form UI.

## Customization

### Projects & Images
Data for projects is located in `src/lib/data.ts`.
To add your own images, place them in the `public/images/` folder and name them as follows (or update the paths in `data.ts`):

- **MediWise**:
  - Main Cover: `public/images/mediwise.png`
  - Gallery 1: `public/images/mediwise-1.png`
  - Gallery 2: `public/images/mediwise-2.png`

- **EcoSwap**:
  - Main Cover: `public/images/ecoswap.png`
  - Gallery 1: `public/images/ecoswap-1.png`
  - Gallery 2: `public/images/ecoswap-2.png`

- **Astro E-commerce**:
  - Main Cover: `public/images/astro.png`
  - Gallery 1: `public/images/astro-1.png`
  - Gallery 2: `public/images/astro-2.png`

- **EMS Dashboard**:
  - Main Cover: `public/images/ems.png`
  - Gallery 1: `public/images/ems-1.png`
  - Gallery 2: `public/images/ems-2.png`
